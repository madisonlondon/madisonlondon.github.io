Projects My Work

Head over to Github to view some of my personal and academic projects!

Some of my favorite projects are:

  • Data Science geoFellowship: I created robust algorithms for data production, focusing on estimating road quality from high resolution satellite data using convolutional neural networks.

  • NASA Grant Forest Disturbances and Ecosystem Services: I worked with a team to create a Tethys web app that supports research on the Amazon watershed and indigenous territories. Ultimately we want to understand how forest degradation, deforestation, and road building affect the ecosystem services provided by the hydrologic cycle in the Southwestern Amazon.

  • Huntington Hill & Webster Apportionments: This Python script outputs the appropriate Huntington-Hill and Webster apportionments given state name and population data. Completed as a final project for our Math & Democracy course.







  • Nucleotide Alignment: This program determines the optimal alignment of two nucleotide sequences. Optimality is determined by the values of the gap penalty g and the weight matrix (that lists all the similarity scores of (ai , bj)). Finally the program outputs the optimal similarity score along with the optimal alignment.

  • Kruskals Algorithm: Using the Swing GUI toolkit, this program implements Kruskalâ€™s algorithm to calculate the minimum spanning tree (MST) of a graph.

  • MLB Salary Data: This program reads data from https://madisonlondon.github.io/mlbsalarydata.github.io/ and uses pandas, a software library, to organize the data and store it in a DataFrame. The program will iterate through the column of salaries, first checking if the value at the current index is not empty. Then, given the condition that we are working with a syntactically correct monetary value, we will remove the dollar sign(s) and comma(s) to store the salary as a float. We find any outliers in the dataset using IQR and then calculate the average.
