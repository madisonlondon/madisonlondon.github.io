My Italian Experience

I began studying Italian in the fall of 2015. Since then I have have taken the STAMP 4s in November 2017, scoring as high as advanced mid.

In the fall of 2019 I went to Verona, Italy, for my semester abroad as a direct enrollment student at the Università degli Studi di Verona. During my semester I completed coursework for both the Livello B1 (intermedio) and Livello B2 (intermedio avanzato).

I have been working as a teaching assistant for the University of Richmond's Italian Department, leading their drill classes for beginner and intermediate intensive courses. The classes give me an opportunity to teach fundamental language, grammar, and speaking skills, while also refreshing them for myself!

My Projects

Head over to Github to view some personal and academic projects!



Some of my favorite projects are:

  • Kruskals Algorithm: Using the Swing GUI toolkit, this program implements Kruskal’s algorithm to calculate the minimum spanning tree (MST) of a graph.

  • Nucleotide Alignment: This program determines the optimal alignment of two nucleotide sequences. Optimality, of course, is determined by the values of the gap penalty g and the weight matrix (that lists all of the s(ai , bj )). Finally the program outputs the optimal similarity score along with the optimal alignment.

  • Student: This program uses inheritance to successfully create a Student class from a Person class.

  • MIPS Binary Encoding: This program reads in a MIPS assembly file specified at the command line. If the file is correct syntactically, each instruction in the file will be translated into its 32 bit MIPS binary encoding and printed to stdout, one per line.
